# Global Command Palette

The **Command Palette** is a powerful navigational tool designed to increase your DevOps speed dramatically. Instead of clicking through 5 different menus to find a specific deployment, you can jump there _instantly_.

## Usage

Press `<Ctrl> + <K>` (or `<Cmd> + <K>` on Mac) from **anywhere** within the dashboard.

## Global Search

The palette will scan through all of your:

- Servers (by Name or IP Address)
- Deployments (by Name or Repository)
- Projects
- Settings & Logs
- Core Documentation

## Keyboard Driven

The palette is entirely keyboard optimized.

- Use the **Up/Down arrow keys** to cycle through results.
- Press **Enter** to instantly navigate to that page.
- Press **Escape** to close the palette.

## Example

1. You notice a server named "Database-Master" is running high on CPU.
2. Press `Ctrl + K`.
3. Type `Data`.
4. Hit `Enter`.
   You are now intimately dropped into the Server Details page for Database-Master within 1 second.
