# Quick Start Guide

Get your first server linked up within 3 minutes.

### Step 1: Add a Server

Go to the **Servers** tab, click 'Add Server'. Enter your VPS IP, Port, Username (usually root), and SSH Key (or Password).

### Step 2: Create a Project

Navigate to **Projects**, click 'New Project'. Assign it to your newly added Server. Input the GitHub Repository URL and the target directory path where you want the code deployed.

### Step 3: Deploy!

Click the 'Deploy' rapid action button. Pulse will SSH into your server, pull the latest code, install dependencies, and run your build setup all through a live streaming terminal.
